Album API: endpoint reference
All responses are in JSON

REQUEST: /
RESPONSE:
    A greeting from this API to you.

REQUEST: /help
RESPONSE:
    This message

REQUEST: /album/{albumName}
RESPONSE:
    A list of all songs in the name of the album (case insensitive). Example: "Cordial"

REQUEST: /temp
RESPONSE:
    A simple "Yay" to confirm that the website is up and running
